#### 5.b heapq 이용
리스트를 미리 만들어놓고 heapq모듈을 하나의 시스템으로 활용할 뿐, 객체생성은 안함.  
힙 구조 만드는데 생성 시간 O(n)으로 위(O(Nlogn))보다 빠름
```python
import heapq
heap = []
heapq.heapify(heap)
heapq.heappush(heap)

```
